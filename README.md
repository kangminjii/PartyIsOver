## 1. 프로젝트 설명
Party Is Over는 멀티 액션 캐주얼 대전게임입니다.   
최대 6인까지 대전할 수 있는 멀티플레이이며, 다양한 아이템과 스킬을 사용해 최후의 1인이 될 때까지 생존하여 승리하는 게임입니다.   

## 2. 개발 환경
- IDE: Visual Studio 2019
- Unity Version: 2021.3.24f1
- VCS: Git, Github
  - GUI: SourceTree
